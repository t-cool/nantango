## Setting up

```bash
$ git clone git@github.com:t-cool/nantango.git
$ cd nantango
$ npm install
```

## Start the app

```bash
$ npm run start
```

## Build the app

```bash
$ npm run build
```

Then the app will be published in `docs` folder.
